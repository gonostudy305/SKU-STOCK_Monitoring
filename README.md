# SKU Stock Monitoring

Static dashboard for the daily inventory risk report.

Live site: <https://gonostudy305.github.io/SKU-STOCK_Monitoring/>

## Overview

This report tracks SKU-level inventory risk for South HCM and North BN warehouses, with focus on:

- ECommerce NMV and unit sales
- Top 30 SKU risk table
- Stock quantity by warehouse
- DOS risk status
- Slow moving main products
- Gift item slow moving stock
- Bingo brand stock and NMV view

## Data Scope

Stock data is based only on BOXME reports:

- Inventory
- Dead Stock

Sales metrics are taken from the processed dashboard/order data used in the report.

- `index.html`: GitHub Pages entrypoint.
